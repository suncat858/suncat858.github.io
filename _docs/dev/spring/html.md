### html 기본틀

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <title>Title of the document</title>
</head>
<body>

<h1>This is a heading</h1>
<p>This is a paragraph.</p>

</body>
</html>
```
### <html>
웹페이지의 시작과 끝
<html>로 시작 </html>로 끝난다

### <head>
웹페이지의 정보,문서에서 사용할 외부 파일들을 링크할때 사용
```
<title>, <meta>등
```
### <body>
브라우저에 실제 표시되는 내용

### <title>
문서 제목

### <meta>
문자 인코딩 및 문서 키워드 요약 정보

### <div>
컨텐츠들을 묶을때 사용한다

### <a>
웹페이지나 외부 사이트 연결

### <script>
코드 삽입

### <link>
외부 파일을 연결할 때 사용한다

### <img>
이미지 삽입
```
<img src="이미지 경로">
```
